💡 What this software does

This tool helps you evaluate mutual funds. It gathers financial data and calculates risk and return metrics using multiple factors. You get a clear view of how funds perform compared to one another, which helps you make informed investment decisions.

The system stores financial data on your own computer. You retain control over your investment information while using automated ranking tools to grade mutual funds.

🛠 System Requirements

  • Windows 10 or 11 operating system.
  • At least 4GB of available RAM.
  • A stable internet connection to fetch current financial data.
  • At least 500MB of free disk space.

🚀 Getting Started

Follow these steps to set up and run the system on your Windows machine.

  1. Visit the download page: Go to the releases page and select the latest installer for Windows.
  2. Install: Run the downloaded installer file. Follow the on-screen prompts to complete the setup process.
  3. Launch: Find the application icon on your desktop or start menu and open it. A command window will open to start the background technical processes.
  4. Access the interface: Open your web browser and navigate to the local address provided in the command window. This will open the dashboard for analysis.
